## User Roles & Permissions

### Visitor

Public users without authentication.

**Permissions:**

- View products & categories
- Search and filter collections
- View product details

### User (Customer)

Authenticated shoppers using the platform.

**Permissions:**

- Add to cart & wishlist
- Place & track orders
- Manage profile & addresses
- Submit reviews & ratings
- Subscribe & Get Exclusive Offers ğŸ
- Like products
- Communicate directly with the manager

### Employee

Operational staff with limited admin access.

**Permissions:**

- Manage products & stock
- Update order statuses
- View customer info (read-only)
- Moderate reviews & support

### Manager

Full system administrators.

**Permissions:**

- Manage users & employees
- View sales analytics
- Configure system settings
- Approve or remove products, reviews, and messages
- Manage user access by blocking or deleting accounts

- Role-based access control (RBAC)
- Secure authentication & authorization
- Modular and scalable architecture
- Centralized management dashboard
- Optimized for large product catalogs
